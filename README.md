# lsb_image_detect

Labtainer lab: LSB Image Detection.

Students analyze a set of PNG images, inspect LSB statistics, identify the image containing hidden LSB data, extract the hidden message, and verify the suspect image.

## Install

Use the imodule package from the GitHub Release page.

Example:

imodule https://github.com/LQLoc1002/lsb_image_detect/releases/download/v1.0/lsb_image_detect_imodule.tar

Then run:

labtainer -r lsb_image_detect
